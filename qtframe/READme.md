# Linux Install Packages Debian

## Install Qt Framework

```sudo apt-get install qt6-base-dev qt6-tools-dev qtcreator ninja-build```

## Install Cmake 

```sudo apt-get install cmake```

## cmake Running Command 

1. Goto `/build` folder 
2. Run Command `cmake ..` example: ../CMakeLists.txt
3. Run Command `cmake --build .`
4. Run Command `./<this is mentio CMakeLists.txt file name>`
