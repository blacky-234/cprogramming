# C Programming

1. Official Documentation & Repositories

GitHub / GitLab / SourceForge → Search "C library <topic>"

GNU Project → Many standard C libraries are hosted here (gnu.org/software)

Gnulib → Common reusable C code (https://www.gnu.org/software/gnulib/)

2. Package Managers for C

While not as universal as Python/npm, there are tools that help:

vcpkg → Microsoft’s C/C++ library manager (https://vcpkg.io/)

Conan → Popular cross-platform C/C++ package manager (https://conan.io/)

pkgsrc → Another package management system (https://pkgsrc.org/)


3. Learning Sites for C Libraries

Awesome C → A huge curated list of useful C libraries.

LibHunt C → Community-ranked libraries.

C Libraries Wiki → A decent starting point.


4. Your System’s Package Manager

On Linux, many C libraries are prebuilt and can be installed:

sudo apt search lib<name>
sudo apt install libcurl-dev



### curl

If you want to learn more about <curl/curl.h>, that’s part of libcurl — a powerful C library for making HTTP, FTP, and other network requests.

1. Official Documentation (Best Place)

libcurl homepage: https://curl.se/libcurl/

C API reference: https://curl.se/libcurl/c/ — all functions, enums, and structs.

Examples: https://curl.se/libcurl/c/example.html
